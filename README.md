# js demo

> A starter project to demo JS libraries

## Features

Layout

- Navigation
- Pages
- Info section
- Attractive design
- Example logo + favicon

UI components

- Input
- Select
- Modal
- Table
- Icon

Functionality

- Built on Vue
- Code Sandbox integration
- Routing
- Vuex

Libraries

- Vue
- Vuex
- Vue Router
- Axios
- Bulma

## Usage

Clone to the `demo/` folder of your local repository.

Then, install and run:

```bash
npm install
npm run dev
```

View at [http://localhost:8080](http://localhost:8080)
