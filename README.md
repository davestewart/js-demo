# Vue Demo Framework

> A starter template to demo JS libs

## Features

Layout

- Navigation
- Pages
- Attractive design
- Example logo + favicon

UI components

- Input
- Select
- Modal
- Table
- Icon

Functionality

- Built on Vue
- Code Sandbox integration
- Routing
- Vuex

Libraries

- Vue
- Vuex
- Vue Router
- Axios
- Bulma

## Setup

Install and run:

```bash
npm install
npm run dev
```

View at [http://localhost:8080](http://localhost:8080).
